# Post Templates

## Hook styles

### Data hook
- `85% of teams are piloting AI agents. Only 5% have them in production.`
- `One number explains where the AI agent market really is: 85% pilot, 5% production.`

### Contrast hook
- `The AI agent race is no longer about demos. It is about production.`
- `Most teams can demo agents. Very few can deploy them.`

### Trend hook
- `Three signals on X say the same thing: AI agents are moving into production competition.`

## Chinese main post template

[hook]

## 3-post thread structure

### Post 1
- hook
- 1–3 market signals
- thesis

### Post 2
- explain the bottleneck
- permissions / reliability / accountability

### Post 3
- where the market goes next
- enterprise integration / runtime / security

## Card copy patterns

### Pattern A
- Top label: `AI AGENT DAILY BRIEF`
- Headline: `85% PILOT`
- Subheadline: `5% PRODUCTION`
- Footer: `From demo to deployment`

### Pattern B
- Headline: `AI AGENT PRODUCTION GAP`
- Body 1: `Oracle → Agent Studio`
- Body 2: `Cloudflare → Runtime`
- Body 3: `Security → Bottleneck`

### Pattern C
- Chinese headline: `AI Agent 落地断层`
- Data line 1: `85% 在试点`
- Data line 2: `5% 真正上线`
- Footer: `下一步拼落地，不拼演示`
